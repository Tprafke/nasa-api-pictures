# NASA API Pictures

### [Live Site](https://tprafke.github.io/nasa-api-pictures/)

## Description

A simple website built using JavaScript, HTML, and CSS. Allows users to view a selection of pictures from NASA's API. Users can also save their favorite pictures in the browser's local storage.
